# daa4-sorting-algorithms

A set of 4 Sorting Algorithms, written in Java.

### Why was this made

 - Academics
 - Understanding Radix LSD and Counting Sort
 - Exploring
   - Testing and assertions
   - Maven
   - Java's streams

## Algorithms

1. Insertion Sort
2. Merge Sort
3. Tree Sort
4. Radix LSD Base N
